
## Usage

To install:
```bash
./scripts/install.sh
source apk-to-java.rc
```
To run an example:
```bash
cd examples
../scripts/apk-to-java.sh HelloWorld.apk
```

The script apk-to-class.sh takes an APK file ("/a/path/FILE.apk").

It creates a directory to store the results (./FILE/apk and ./FILE/java).
